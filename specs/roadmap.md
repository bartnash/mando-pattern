# Roadmap & Progress

## Current Status (v0.1.0)
- [x] Basic project scaffolding (Vite + React 19).
- [x] Movable chord transposition logic.
- [x] Infinite carousel navigation with swipe gestures.
- [x] Responsive SVG chord diagrams.
- [x] PWA support & Offline functionality.
- [x] Custom branding/icons from `mando-shapes`.
- [x] Refactored Spec-Driven Development architecture.

  - [x] Add new pattern, "d-chop", which used the 4 chord from the g-chop pattern as its root.
  - [ ] Provide ability to "play" through the chord progression
  - [ ] Audio playback for individual chords.
  - [ ] Custom pattern builder (User persistence).
- **Long Term**
  - [ ] Alternative tunings support (Octave Mandolin, Mandocello).
  - [ ] Social sharing of pattern links.
